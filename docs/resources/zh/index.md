## 工具

### 开发

- **VS Code** - 推荐的编辑器，配合 Vue 扩展
- **Volar** - VS Code 的 Vue 语言支持
- **pnpm** - 快速、节省磁盘空间的包管理器

### 部署

- **GitHub Pages** - 静态站点的免费托管
- **Vercel** - 零配置部署
- **Netlify** - 持续部署平台
