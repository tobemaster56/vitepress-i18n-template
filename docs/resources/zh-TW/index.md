## 工具

### 開發

- **VS Code** - 推薦的編輯器，配合 Vue 擴充功能
- **Volar** - VS Code 的 Vue 語言支援
- **pnpm** - 快速、節省磁碟空間的套件管理器

### 部署

- **GitHub Pages** - 靜態網站的免費託管
- **Vercel** - 零配置部署
- **Netlify** - 持續部署平台
