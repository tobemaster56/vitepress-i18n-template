# 文件概述

歡迎閱讀文件！這是一個用於創建多語言文件的 VitePress 模板。

## 功能特點

本模板包含：

- **多語言支援** - 英文、簡體中文和繁體中文
- **版本化結構** - 按產品版本組織文件
- **自訂組件** - 語言切換器和 Markdown 容器
- **現代工具鏈** - 基於 VitePress 2.0 和 Vue 3.5 構建

## 快速連結

- [快速開始](./getting-started.md) - 了解如何使用此模板
- [配置](./configuration.md) - 配置您的文件網站

## 自訂容器

本模板包含自訂 Markdown 容器：

::: tip 提示
這是一個用於顯示有用資訊的提示容器。
:::

::: warning 警告
這是一個用於顯示重要通知的警告容器。
:::

::: danger 危險
這是一個用於顯示關鍵警告的危險容器。
:::

::: details 點擊展開
這是一個可摺疊的詳情容器。
:::
