# 文档概述

欢迎阅读文档！这是一个用于创建多语言文档的 VitePress 模板。

## 功能特点

本模板包含：

- **多语言支持** - 英文、简体中文和繁体中文
- **版本化结构** - 按产品版本组织文档
- **自定义组件** - 语言切换器和 Markdown 容器
- **现代工具链** - 基于 VitePress 2.0 和 Vue 3.5 构建

## 快速链接

- [快速开始](./getting-started.md) - 了解如何使用此模板
- [配置](./configuration.md) - 配置您的文档站点

## 自定义容器

本模板包含自定义 Markdown 容器：

::: tip 提示
这是一个用于显示有用信息的提示容器。
:::

::: warning 警告
这是一个用于显示重要通知的警告容器。
:::

::: danger 危险
这是一个用于显示关键警告的危险容器。
:::

::: details 点击展开
这是一个可折叠的详情容器。
:::
