# Documentation Overview

Welcome to the documentation! This is a template for creating multi-language documentation with VitePress.

## Features

This template includes:

- **Multi-language support** - English, Simplified Chinese, and Traditional Chinese
- **Version-based structure** - Organize docs by product versions
- **Custom components** - Language switcher and markdown containers
- **Modern tooling** - Built with VitePress 2.0 and Vue 3.5

## Custom Containers

This template includes custom markdown containers:

::: tip
This is a tip container for helpful information.
:::

::: warning
This is a warning container for important notices.
:::

::: danger
This is a danger container for critical warnings.
:::

::: details Click to expand
This is a collapsible details container.
:::
